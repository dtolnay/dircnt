Count directory entries faster than `ls -f | wc -l`.

### Installation

```console
$ cargo install dircnt
```

<br>

### Usage

```console
$ dircnt path/to/dir
63496110
```

Like `ls | wc -l` but unlike `ls -f | wc -l`, the special directory entries `.`
and `..` are excluded from the count.
